# 01 Getting Started

This module introduces the basics of C++ and how to compile a simple program.

## What is C++?

C++ is a powerful programming language used to build fast and efficient applications.
It is a compiled language, which means the code is converted into machine code by a compiler before it runs.
C++ supports object-oriented programming, procedural programming, and generic programming.

## What type of language is C++?

C++ is a general-purpose, statically typed language.
It gives you control over memory and performance, while also supporting high-level abstractions like classes and objects.

## How does C++ work?

C++ code is written in source files, then compiled into an executable program.
The compiler checks the code, translates it into machine instructions, and the operating system runs the resulting program.

## Why learn C++?

C++ is widely used in many important areas:
- Game development
- System software and operating systems
- Embedded systems
- High-performance applications
- Simulation and scientific computing

## Why C++ is better for some tasks

C++ is often chosen when speed and efficient use of hardware matter.
It is faster than many interpreted languages because of compilation.
It also offers more control over memory and low-level system behavior than some newer languages.
